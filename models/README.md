# 🚨 Tigrinya Harmful Content Detection System

A machine learning-based system for detecting **harmful vs neutral Tigrinya text** using an XGBoost model, FastAPI backend, and Streamlit dashboard.

---

## 🧠 Project Overview

This project classifies Tigrinya (ትግርኛ) text into:

- 🔴 Harmful Content
- 🟢 Neutral Content

It is built as a **baseline AI moderation system** for social media safety.

⚠️ Important Note:
This model is trained on a **small sample dataset**, so results are **not production-ready**. Performance can be significantly improved with:
- Larger labeled datasets
- Better Tigrinya preprocessing
- Advanced NLP models (e.g., Transformers)

---

## ⚙️ Tech Stack

- 🧠 Machine Learning: XGBoost
- 🔤 Feature Engineering: TF-IDF + keyword features
- 🚀 Backend: FastAPI
- 🎨 Frontend: Streamlit
- 🐍 Language: Python
